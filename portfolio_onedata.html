NOTEBOOK 01 · CORE COMPETENCY

Data Quality Assessment & Remediation

Automated profiling pipeline that checks completeness, validity, uniqueness, and freshness across 50K+ retail records. Generates a quality score, triggers a pipeline gate, and runs a remediation workflow with full audit trail. Raised quality score from 72 → 93 before downstream processing.

The profiler class, pipeline gate logic, and remediation audit log are production-ready patterns applicable across any data product context.

quality_gate.py
def gate(self, min_score=85):
  score = self.overall_score()
  status = '✅ PASS' if score >= min_score
         else '🚫 BLOCK'
  return score, status
NOTEBOOK 02

Demand Forecasting Data Product

End-to-end forecasting pipeline: feature engineering (lag, seasonal, weather, promo), 3-model comparison (Random Forest, GBM, Ridge) on time-series split, versioned output file matching the data contract schema. MAPE reduced from 18.4% → 10.7%.

NOTEBOOK 03

Geospatial Demand Signals

K-Means clustering of 1,500+ store locations across 7 European markets. Elbow method + silhouette score for optimal k. Identified 4 store archetypes (Urban Hub, Suburban, Rural, Outlet) — reducing forecast MAPE by ~5pp when used as model stratifier.
